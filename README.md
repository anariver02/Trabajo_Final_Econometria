# Trabajo_Final_Econometria
# Predicción de la eficiencia energética de edificios residenciales

Este repositorio contiene los materiales asociados al trabajo de Econometría sobre predicción de la carga de calefacción en edificios residenciales.

## Contenido

- Documento final del trabajo.
- Resumen en inglés.
- Base de datos original.
- Particiones train/test utilizadas.
- Notebooks de Google Colab para MCO, Lasso, Eco-RETINA, Random Forest y Red Neuronal.
- Tablas y figuras generadas.

## Modelos estimados

- Mínimos Cuadrados Ordinarios
- Lasso
- Eco-RETINA
- Random Forest
- Red Neuronal

## Objetivo

Comparar el rendimiento predictivo y la interpretabilidad de distintos modelos econométricos y de aprendizaje automático para estimar la carga de calefacción de edificios residenciales.
